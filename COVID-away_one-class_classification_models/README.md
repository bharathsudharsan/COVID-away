i. Acc, ii. Acc+Gyro, iii. Acc+Gyro+Mbar, iv. Acc+Gyro+Mbar+RVect.

i. iForest.py, ii. Local_Outlier_Factor.py, iii. Minimum_Covariance_Determinant.py, iv. OC_SVM.py.
